# 버튼 앱
### TDD 기반 버튼 토이 프로젝트
##  프로젝트 설치 및 시작

#### 프로젝트 클론

```shell
$ git clone https://github.com/YSBINN/TDD-button-app.git
```

#### 패키지 설치

```shell
$ yarn or yarn install
```

#### 서버 실행

```shell
$ yarn start
```

## 구현기능
 [**Testing Library**](https://testing-library.com/) 사용 해서 DOM, User-Action, Style 테스트 케이스 코드 작성 및 간단한 버튼 UI 구현
